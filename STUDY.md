# 📖 메뉴 추천 서비스 개발 가이드 (Study Guide)

이 프로젝트는 입문자도 쉽게 수정할 수 있도록 **HTML, CSS, JavaScript** 세 가지 파일로 구성되어 있습니다. 각 파일을 어떻게 수정하고 발전시킬 수 있는지 핵심 내용을 정리해 드릴게요!

---

## 1. 프로젝트 구조 이해하기

- **`index.html`**: 웹사이트의 **뼈대**입니다. 버튼, 글자, 결과창 등 눈에 보이는 요소들을 배치합니다.
- **`style.css`**: 웹사이트의 **옷**입니다. 색상, 크기, 간격, 애니메이션 등 디자인을 담당합니다.
- **`app.js`**: 웹사이트의 **두뇌**입니다. 메뉴 데이터 저장, 랜덤 선택 로직, 버튼 클릭 이벤트 등을 처리합니다.

---

## 2. 직접 수정해보기 (실습)

### ✅ 메뉴를 더 추가하고 싶을 때 (`app.js`)
`menus` 객체 안에 원하는 메뉴를 `{ name: '이름', emoji: '아이콘' }` 형식으로 추가하기만 하면 됩니다.
```javascript
const menus = {
    lunch: [
        { name: '새로운 메뉴', emoji: '✨' }, // 이렇게 추가하세요!
        // ... 기존 메뉴들
    ],
    // ... dinner 도 동일
};
```

### ✅ 배경색이나 포인트 색상을 바꾸고 싶을 때 (`style.css`)
파일 맨 위쪽의 `:root` 부분을 수정하면 전체적인 테마 색상을 한 번에 바꿀 수 있습니다.
```css
:root {
    --primary-color: #ff6b6b; /* 이 코드를 다른 색상 코드로 바꿔보세요 */
    --bg-color: #f8f9fa;      /* 배경색을 바꾸고 싶다면 여기를 수정하세요 */
}
```

### ✅ 로딩 시간을 조절하고 싶을 때 (`app.js`)
`setTimeout` 함수 뒤의 숫자(밀리초 단위, 1000 = 1초)를 수정하면 됩니다.
```javascript
setTimeout(() => {
    // ... 로직
}, 800); // 800을 2000으로 바꾸면 2초 동안 "고민 중" 상태가 유지됩니다.
```

## 3. 나중에 도전해볼만한 기능 (Next Steps)

1. **카테고리 추가**: '야식'이나 '디저트' 카테고리를 추가해 보세요. (`index.html`에 버튼 추가 + `app.js`에 데이터 추가)
2. **애니메이션 변경**: `style.css`의 `@keyframes`를 수정해서 메뉴가 나타날 때 효과를 바꿔 보세요.
3. **히스토리 기능**: 내가 지금까지 추천받은 메뉴들을 아래에 리스트로 쌓아주는 기능을 만들어 보세요.

**Tip:** 코드를 수정한 후에는 브라우저에서 `index.html`을 새로고침하여 결과를 바로 확인할 수 있습니다. 직접 이것저것 바꿔보면서 감을 익히는 것이 가장 빠른 학습 방법입니다! 💪
